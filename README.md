# How To
## Stap Install
- npm install readline-sync
- npm install request
- npm install colors
- npm install librcon
- npm install utf8

## Stap Run
- node index.php

## Stap Use Program
- yes/no For Run Program
- inport ip [ Don't port ]
- rcon port
- rcon password
- Done.
