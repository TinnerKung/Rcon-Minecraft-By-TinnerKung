# How To
## Stap Install
```js
$ npm install readline-sync
$ npm install request
$ npm install colors
$ npm install librcon
$ npm install utf8
```

## Stap Run
```sh
- node index.php
```

## Stap Use Program
```js
- yes/no For Run Program
- inport ip [ Don't port ]
- rcon port
- rcon password
- Done.
```
